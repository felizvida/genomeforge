Professional User Training Tutorial

Training Flow

  1. Foundation: sequence loading, map, and sequence track navigation.
  2. Experimental design: digest, primers, PCR, gel, and cloning simulation.
  3. P0 interop: DNA container import/export and AB1 trace workflows.
  4. P1 analysis: primer specificity, ranking, CRISPR, and HDR donor design.
  5. P2 governance: permissions, audit logs, project diff, and review approval.
  6. Operational completion: project persistence, sharing, and final checks.

1. Foundation Workflow

1.1 Load and Validate Sequence

  1. Paste FASTA/GenBank/EMBL/raw DNA into Sequence / FASTA / GenBank.
  2. Set Name, Topology, and click Refresh Info.
  3. Confirm stats: length, GC%, topology, feature count.

1.2 Render Map and Inspect Features

  1. Open Map tab and set enzymes (for example EcoRI,BamHI,HindIII).
  2. Click Render Map and interact with cuts/features.
  3. Use zoom/pan and read details from the Selection Inspector.

Figure 1: Restriction map with interactive feature and cut inspection.

2. Sequence Track and Alignment Views

2.1 Sequence Track Navigation

  1. Set track start/end and frame, then click Render Sequence Track.
  2. Use minimap brush to move/resize window for large sequences.
  3. Click codons for residue mapping and feature context.

2.2 MSA Heatmap

  1. In Advanced, add multiple sequences and run Run MSA.
  2. Click MSA Identity Heatmap.
  3. Use consensus and phylogeny tools for sequence comparison.

Figure 2: Sequence track with translated frame lane and feature overlays.

Figure 3: Identity heatmap generated from multiple sequence alignment.

3. Core Experimental Workflows

3.1 Primers, PCR, and Gel

  1. Design primers in Primers/PCR using target region fields.
  2. Run Virtual PCR and inspect product sizes.
  3. Run PCR Gel Lanes to visualize expected bands against marker ladder.

3.2 Cloning and Ligation

  1. Use Gibson / Golden Gate / Gateway / TOPO / TA-GC tools in Advanced.
  2. Run Simulate Ligation Products and inspect ranked pathways.
  3. Apply graph filters (all, desired, byproducts) and inspect top candidates.

Figure 4: Ligation pathway graph with probability-weighted product ranking.

4. P0: Interop and AB1 Trace Training

4.1 DNA Container Import/Export

  1. Open Trace/Interop tab.
  2. Click Export DNA Container to generate base64 package.
  3. Paste base64 into DNA Container and click Import DNA Container.
  4. Confirm sequence/name/topology and feature restoration.

4.2 AB1 Trace Workflow

  1. Paste AB1 base64 (or use synthetic sequence fallback) and click Import AB1.
  2. Run Trace Summary to review quality metrics.
  3. Provide reference and run Trace Align.
  4. Edit a base using Edit Trace Base, then run Trace Consensus.
Operational Note: Current interop supports Genome Forge DNA container robustly. Native proprietary SnapGene binary parsing remains partial.

5. P1: Primer Specificity and CRISPR Helpers

5.1 Primer Specificity and Ranking

  1. In Advanced, provide background sequences (one per line).
  2. Click Primer Specificity to estimate off-target product risk.
  3. Provide candidate JSON and click Rank Primer Pairs.

5.2 CRISPR and HDR

  1. Set PAM/spacer settings and run gRNA Design.
  2. Select a guide and run CRISPR Off-target.
  3. Configure edit window and homology arms, then run Design HDR Donor.
Scientific Caution: P1 scoring is heuristic and intended for design triage. Validate final candidates with independent wet-lab and genome-aware tools.

6. P2: Collaboration, Audit, and Review

  1. Create workspace with owner/members using Create Workspace.
  2. Assign reviewer role via Set Permission.
  3. Run Get Audit Log after save/delete events for traceability.
  4. Use Project Diff to compare project states.
  5. Run Submit Review, then approve using Approve Review.
Recommended Governance Pattern: editor submits review, reviewer/owner approves, and audit log is exported into release evidence.

Figure 5: History and review-ready version tracking context.

7. End-to-End Lab Exercise

  1. Load sequence and annotate/inspect map.
  2. Design primers, run PCR, and simulate gel lanes.
  3. Run cloning compatibility and ligation simulation.
  4. Export DNA container and re-import.
  5. Run AB1 trace align/edit/consensus workflow.
  6. Run primer specificity and CRISPR + HDR design.
  7. Save project, assign reviewer role, submit and approve review.
  8. Create share bundle and verify /share/<id> output.
