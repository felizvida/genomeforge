1) Training Dataset (Real-World)

Use the bundled dataset files under docs/tutorial/datasets/:

  • training_real_world_sequences.fasta (EGFP, mCherry, pUC19 MCS, BRAF exon 15 region, lacZ alpha fragment).
  • training_real_world_dataset.json (structured records, enzyme panels, primer candidates, CRISPR seeds, and collaboration actors).
  • case_playbook.md (scenario prompts and expected outputs).
Why this dataset: it combines common cloning inserts, reporter genes, and a medically relevant target region, allowing realistic analysis without requiring private lab data.

2) Case A: Restriction Mapping and Track Interpretation

A1. Load pUC19 MCS Training Record

  1. Copy pUC19_MCS from dataset JSON or FASTA.
  2. Set topology to circular.
  3. Run Refresh Info, then Render Map.

A2. Digest Interpretation

  1. Use enzymes EcoRI,BamHI,HindIII,XbaI,PstI,KpnI.
  2. Run Digest and record fragment pattern.
  3. Run Digest (Methylation Aware) with motif list to compare blocked cuts.

3) Case B: Primer Design, Specificity, and PCR Outcome

B1. Primer Generation

  1. Load EGFP CDS from dataset.
  2. Set target window and run Design Primers.
  3. Run Primer Diagnostics and capture Tm/risk flags.

B2. Specificity Ranking

  1. Paste background panel (EGFP + mCherry + BRAF fragment).
  2. Run Primer Specificity.
  3. Run Rank Primer Pairs using candidate JSON from dataset.

B3. PCR and Gel Lanes

  1. Run Virtual PCR and verify predicted product size.
  2. Run PCR Gel Lanes with 1kb_plus ladder.
  3. Compare expected and observed in-silico band positions.

4) Case C: Cloning Strategy and Ligation Product Ranking

  1. Use dataset vector/insert examples for Gateway/TOPO/TA-GC trials.
  2. Run Cloning Compatibility and document reasoned pass/fail.
  3. Run Simulate Ligation Products with byproducts enabled.
  4. Inspect pathway graph and prioritize highest probability, in-frame product.

5) Case D: P0 Interop and AB1 Trace Operations

D1. DNA Container Roundtrip

  1. Export current record via Export DNA Container.
  2. Re-import via Import DNA Container.
  3. Verify name/topology/length/features are preserved.

D2. AB1 Trace Routine

  1. Import AB1 payload (or use synthetic seed sequence from dataset).
  2. Run Trace Summary, then Trace Align to EGFP reference.
  3. Edit one low-confidence base and regenerate Trace Consensus.
Interop boundary: Genome Forge DNA container is robust. Native proprietary SnapGene binary parsing remains limited in this release.

6) Case E: P1 CRISPR and HDR Workflow

  1. Load BRAF exon 15 fragment from dataset.
  2. Run gRNA Design with PAM NGG, spacer length 20.
  3. Select top guide and run CRISPR Off-target across background panel.
  4. Configure edit coordinates and run Design HDR Donor.
Capture: selected guide, off-target risk score, donor length, and PAM disruption suggestions for review package.

7) Case F: P2 Collaboration, Audit, and Approval

  1. Create workspace with owner, editor, reviewer from dataset actors.
  2. Assign reviewer role through Set Permission.
  3. Save project snapshot and query Get Audit Log.
  4. Use Project Diff for current vs baseline comparison.
  5. Run Submit Review and Approve Review.

8) Capstone Exercise (All Features)

Objective: deliver a review-approved cloning package with evidence trail.

  1. Map + digest + track inspection for chosen construct.
  2. Primer generation + specificity ranking + PCR lane prediction.
  3. Assembly/ligation pathway selection.
  4. DNA container export/import verification.
  5. Trace align/edit/consensus check.
  6. CRISPR candidate + off-target + HDR donor design.
  7. Save project, assign reviewer, submit and approve review.
  8. Create share bundle and archive artifact IDs.
