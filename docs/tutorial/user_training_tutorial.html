Genome Forge: Step-by-Step User Training Tutorial

This guide teaches core daily workflows in the Genome Forge web app, from sequence loading and map inspection through cloning simulation and project sharing.

Prerequisite: start the app with python3 web_ui.py --port 8080 and open http://127.0.0.1:8080.

Contents

  1. Load a sequence and check basic info
  2. Render plasmid map with cut sites
  3. Inspect sequence track and translation
  4. Run alignment and heatmap analysis

  6. Assess star activity risk
  7. Simulate ligation products
  8. Primer/PCR and gel lane simulation
  9. Save projects, collections, and shares

Step 1. Load a Sequence and Check Basic Info

  1. Paste FASTA/GenBank/EMBL/raw DNA into the Sequence / FASTA / GenBank box.
  2. Set Name and Topology.
  3. Click Refresh Info to confirm length, GC, and feature count.

Tip: if you import EMBL, Genome Forge parses both sequence and feature lines.

Step 2. Render Plasmid Map with Restriction Cuts

  1. Open Map tab.
  2. Enter enzymes (for example EcoRI,BamHI).
  3. Click Render Map.
  4. Use mouse wheel to zoom and drag to pan.
  5. Click feature arcs or cut markers to inspect details in the inspector panel.

Step 3. Inspect Sequence Track and Translation

  1. Set trackStart/trackEnd and translation frame.
  2. Click Render Sequence Track.
  3. Use the minimap brush to move and resize the view window.
  4. Click codon cells to inspect residue mapping.

Step 4. Run Alignment and Identity Heatmap

  1. Open Advanced tab and provide multiple sequences.
  2. Select MSA method (progressive or installed external aligners).
  3. Click Run MSA, then MSA Identity Heatmap.

Step 5. Review Project History Graph

  1. Save your current sequence as a project.
  2. Perform edits and save additional snapshots.
  3. Click Project History Graph to render version nodes.
  4. Recency coloring highlights newer versions.

Step 6. Assess Star Activity Risk

  1. Set enzymes and choose star activity level.
  2. Run Star Activity Scan.
  3. Review off-target burden by enzyme and click top cuts to center sequence track.

Step 7. Simulate Ligation and Rank Products

  1. Provide vector/insert sequences and enzyme ends.
  2. Enable byproducts for realistic reaction scenarios.
  3. Run Simulate Ligation Products.
  4. Use graph filters and inspect product probabilities and junction diagnostics.

Step 8. Primer/PCR and Gel Lanes

  1. Design primers around a target region.
  2. Run Virtual PCR.
  3. Use PCR Gel Lanes with selected marker ladder to visualize expected product bands.

Step 9. Save Projects, Collections, and Shares

  1. Save projects from current sequence state.
  2. Group related projects into a collection.
  3. Create a share bundle and copy the generated /share/<id> URL.

This enables portable handoff and quick viewer access later.

Recommended Practice Sequence

  1. Map + digest analysis
  2. Track + translation inspection
  3. Primer design + PCR + gel
  4. Cloning compatibility + ligation ranking
  5. Save project + collection + share bundle
