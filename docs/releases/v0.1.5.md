# Genome Forge v0.1.5

Release date: 2026-04-02

## Highlights

- Broadened ambiguity-aware DNA behavior from simple input preservation to actual matching semantics across the product.
- Motif search and PCR simulation now treat IUPAC codes as compatible nucleotide sets rather than exact string literals.
- BLAST-like local search and primer-specificity reporting now score ambiguity-compatible matches correctly.
- Added regression coverage to protect the new behavior and fixed a related mixed text/DNA search regression in `/api/search-entities`.
