# Genome Forge v0.1.1

Release date: 2026-03-04

## Highlights

1. Completed project-wide rebrand cleanup to remove legacy SnapGene naming in active code and documentation.
2. Expanded and reorganized training tutorial to 20 biologically clustered cases with richer interpretation guidance.
3. Refreshed handoff and feature-coverage documentation for consistent Genome Forge naming.

## What's New

### Documentation and Training

1. Tutorial expanded to 20 cases (A-T) with biological-meaning clusters.
2. Added case-selection matrix and interpretation playbook for CS-first users.
3. Strengthened biology/context explanations and input-data detail per case.
4. Regenerated tutorial PDF from updated HTML for synchronized output.

### Naming and Consistency

1. Renamed core CLI module:
   - `snapgene_like.py` -> `genomeforge_toolkit.py`
2. Renamed feature coverage document:
   - `SNAPGENE_FEATURE_COVERAGE.md` -> `FEATURE_COVERAGE.md`
3. Updated internal imports and references accordingly.

### Compatibility Notes

1. Native proprietary `.dna` binary parsing remains partial.
2. Genome Forge DNA container import/export remains the recommended interop path.
