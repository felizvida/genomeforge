# Genome Forge v0.1.2

Release date: 2026-03-04

## Highlights

- Added advanced sequence-analysis and workflow features:
  - BLAST-like local nucleotide search (`/api/blast-search`)
  - Sanger chromatogram rendering (`/api/trace-chromatogram-svg`)
  - Trace-based verification and genotyping (`/api/trace-verify`)
  - Reference element libraries + auto-flag scanning (`/api/reference-db-*`, `/api/reference-scan`)
  - siRNA candidate design and mapping (`/api/sirna-design`, `/api/sirna-map`)
- Expanded Web UI for these workflows in Trace/Interop and Advanced tabs.
- Expanded tests to cover all new APIs and workflows.
- Updated training tutorial substantially:
  - Increased to 37 cases (A-AK)
  - Added new cases for chromatogram, genotyping, BLAST-like search, and reference/siRNA workflows
  - Added per-case sections:
    - Step-by-step software guide
    - Sample results
    - Expected results
    - Interpretation guidance
    - Biological explanation
  - Regenerated PDF tutorial.
