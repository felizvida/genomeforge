# Genome Forge v0.1.3

Release date: 2026-03-16

## Highlights

- Fixed the reference auto-flag workflow so newly detected elements are returned to the UI and remain visible in the active session.
- Fixed BLAST-like local search coverage reporting so partial matches no longer appear as full-length coverage hits.
- Fixed chromatogram rendering to use raw AB1 sample positions when available, improving real trace visualization.
- Strengthened smoke and real-world functional tests to cover these regression paths directly.
