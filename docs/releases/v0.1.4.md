# Genome Forge v0.1.4

Release date: 2026-03-23

## Highlights

- Completely rebuilt the training tutorial around richer biological meaning, clearer data explanation, and stronger interpretation guidance.
- Added real UI screenshots for flagship workflows directly into the HTML/PDF tutorial and the README.
- Added a source-driven tutorial generator plus a reusable screenshot-capture script so the training package can be regenerated instead of hand-edited.
- Added a dataset guide and prebuilt sample-data bundles for all 37 tutorial cases to make the training data easier to use immediately.
