# Genome Forge v0.1.6

Release date: 2026-04-02

## Highlights

- Expanded the training course from 37 to 39 lessons.
- Added explicit ambiguity-aware teaching with new lessons on degenerate primers and identity search on uncertainty-bearing sequences.
- Gave the tutorial a textbook-style facelift with stronger front matter, better print rhythm, and more polished screenshot coverage.
- Refreshed the generated HTML/PDF tutorial package, case playbook, dataset metadata, and new per-case bundles for the added lessons.
