# Genome Forge v0.1.0

Release date: 2026-03-04

## Highlights

1. Genome Forge web workbench with broad cloning and analysis workflow coverage.
2. Modernized interactive UI with map/track exploration, minimap, and rich visualization panels.
3. Canonical schema conversion and record roundtripping.

## New in This Release

### P0 Interop + Trace Core

1. DNA container import/export APIs:
   - `/api/import-dna`
   - `/api/export-dna`
2. AB1/trace workflow foundation:
   - `/api/import-ab1`
   - `/api/trace-summary`
   - `/api/trace-align`
   - `/api/trace-edit-base`
   - `/api/trace-consensus`
3. New UI `Trace/Interop` tab for practical usage.

### P1 Primer Specificity + CRISPR Helpers

1. Primer specificity/off-target scanning and ranking:
   - `/api/primer-specificity`
   - `/api/primer-rank`
2. CRISPR helper endpoints:
   - `/api/grna-design`
   - `/api/crispr-offtarget`
   - `/api/hdr-template`
3. Advanced-tab UI controls for primer and CRISPR workflows.

### P2 Collaboration + Review Core

1. Workspace creation and per-project role map:
   - `/api/workspace-create`
   - `/api/project-permissions`
2. Audit trail and structured diff/review:
   - `/api/project-audit-log`
   - `/api/project-diff`
   - `/api/review-submit`
   - `/api/review-approve`
3. Review permission gate for approval path (`reviewer` or `owner` role).
